### 📥 Installation and run
#### Clone the repo
```sh
git clone https://github.com/flor4/holbertonschool-printf
```
#### Enter in the folder with
```sh
cd holbertonschool-printf
```
#### Compile with 
```sh
gcc *.c
```
## 💡 Usage
use `_printf()` in c file.
